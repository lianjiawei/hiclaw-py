Quickstart

快速安装，清晰配置。

1

Install

curl -fsSL https://raw.githubusercontent.com/lianjiawei/hiclaw-py/master/scripts/install.sh | bash

Linux / macOS / WSL2 自动克隆、创建 venv、安装 CLI，并构建 Dashboard。

2

Configure

hiclaw setup

交互式创建 `.env`，引导配置 Provider、Telegram / Feishu 和 Dashboard。

3

Operate

hiclaw doctor hiclaw start hiclaw status hiclaw logs -f

用于服务器长期运行；本地调试可以使用 `hiclaw run` 或 `hiclaw-tui`。

Uninstall

卸载也同样简单。

HiClaw 提供一键卸载脚本，可移除安装目录、虚拟环境与命令包装器。 如果你希望保留 `.env`、`data/`、`workspace/` 等本地数据，也支持保留模式卸载。

1

Linux / macOS / WSL2

curl -fsSL https://raw.githubusercontent.com/lianjiawei/hiclaw-py/master/scripts/uninstall.sh | bash
2

Windows PowerShell

irm https://raw.githubusercontent.com/lianjiawei/hiclaw-py/master/scripts/uninstall.ps1 | iex
3

Keep local data

HICLAW_KEEP_DATA=1 curl -fsSL https://raw.githubusercontent.com/lianjiawei/hiclaw-py/master/scripts/uninstall.sh | bash

适合希望移除程序本体，但保留配置和运行数据的场景。

Core Capabilities

围绕企业运营场景构建的智能体能力矩阵。

多智能体协同决策

规划、执行、复核与用户自定义 Agent 可按角色协同，支撑复杂任务分工、流程闭环与责任追踪。

行业运营自动化

覆盖定时提醒、巡检记录、日报生成、风险排查和长期任务运行，降低重复性工作的人力成本。

可观测运行看板

实时展示智能体状态、授权等待、工具调用与任务进展，让关键流程更透明、可管控、可审计。

稳定的部署体验

从安装、配置到启动前检查形成完整闭环，用更友好的诊断提示降低上线与运维门槛。

多通道接入

支持 Telegram、Feishu、TUI 与后台服务模式，适配企业内部协作、值班响应与本地调试场景。

安全与可控边界

通过工具授权、工作区限制、日志追踪与状态命令，帮助企业在效率提升的同时保持运行可控。

Operational Visibility

让智能体运行状态真正可视、可查、可追踪。

HiClaw 的 Pixel Office Dashboard 将抽象任务转化为直观运行视图， 帮助团队快速识别工作中、等待中、阻塞中、已完成与空闲状态，提升运营响应效率与管理透明度。

Architecture

面向长期运行与企业落地的统一智能体底座。

Single Agent

轻量任务走单智能体路径，保证更低延迟与更直接的响应效率。

Agent Cluster

复杂任务由 planner、executor、reviewer 按计划拆解协同，支持更稳健的执行与复核闭环。

Custom Agents

企业可基于能力目录扩展自定义 Agent，将行业知识、岗位职责与业务流程沉淀为专属角色能力。
